    <h2>1. 概述</h2>

    <p>SoulCards（「我們」、「本應用」）重視您的隱私權。本隱私權政策說明我們如何收集、使用、儲存和保護您在使用 SoulCards 塔羅占卜應用程式時的個人資訊。</p>


    <h2>2. 我們收集的資訊</h2>

    

    <h3>2.1 帳戶資訊</h3>

    <ul>

        <li>電子郵件地址（用於帳戶註冊和登入）</li>

        <li>用戶名稱和個人資料設定</li>

        <li>頭像圖片（可選）</li>

    </ul>


    <h3>2.2 占卜記錄</h3>

    <ul>

        <li>您的占卜問題和選擇的占卜類型</li>

        <li>抽取的塔羅牌結果</li>

        <li>AI 生成的解讀內容</li>

        <li>占卜時間和歷史記錄</li>

    </ul>


    <h3>2.3 設備和使用資訊</h3>

    <ul>

        <li>設備識別碼（用於同步數據）</li>

        <li>應用程式使用統計</li>

        <li>錯誤報告和診斷資訊</li>

    </ul>


    <h3>2.4 訂閱和付款資訊</h3>

    <ul>

        <li>Apple App Store 交易記錄</li>

        <li>訂閱狀態和到期日期</li>

        <li>購買歷史（通過 Apple IAP 處理）</li>

    </ul>


    <h2>3. 資訊使用方式</h2>

    <p>我們使用收集的資訊來：</p>

    <ul>

        <li>提供塔羅占卜服務和 AI 解讀</li>

        <li>維護您的帳戶和個人化體驗</li>

        <li>處理訂閱和付款</li>

        <li>改善應用程式功能和用戶體驗</li>

        <li>提供客戶支援服務</li>

    </ul>


    <h2>4. 第三方服務</h2>

    

    <h3>4.1 Supabase（數據庫和認證）</h3>

    <ul>

        <li>安全儲存您的帳戶資訊和占卜記錄</li>

        <li>提供跨設備數據同步</li>

        <li>符合 GDPR 和 SOC 2 標準</li>

    </ul>


    <h3>4.2 OpenRouter/Google Gemini（AI 服務）</h3>

    <ul>

        <li>處理占卜解讀請求</li>

        <li>不會永久儲存您的個人問題</li>

        <li>僅用於生成個人化解讀內容</li>

    </ul>


    <h3>4.3 Apple App Store（付款處理）</h3>

    <ul>

        <li>處理所有訂閱和購買交易</li>

        <li>我們不會直接處理您的付款資訊</li>

        <li>遵循 Apple 的隱私權政策</li>

    </ul>


    <h2>5. 數據保護和安全</h2>

    <ul>

        <li>所有數據傳輸均使用 HTTPS 加密</li>

        <li>密碼使用安全雜湊演算法儲存</li>

        <li>定期進行安全性更新和監控</li>

        <li>遵循業界最佳實務標準</li>

    </ul>


    <h2>6. 您的權利</h2>

    <p>根據適用的隱私權法律，您有權：</p>

    <ul>

        <li>查看我們持有的您的個人資料</li>

        <li>要求更正不準確的資訊</li>

        <li>要求刪除您的帳戶和資料</li>

        <li>撤回同意（在適用情況下）</li>

        <li>數據可攜性（匯出您的資料）</li>

    </ul>


    <h2>7. 數據保留</h2>

    <ul>

        <li>帳戶資訊：直到您刪除帳戶</li>

        <li>占卜記錄：直到您手動刪除或帳戶刪除</li>

        <li>使用統計：匿名化後保留用於改善服務</li>

    </ul>

    <p>本應用程式適用於 13 歲以上用戶。我們不會故意收集 13 歲以下兒童的個人資訊。</p>
